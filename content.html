Example solution

We can start by using the max() function to find the largest value in the list. When abs_value is True, the code compares the positive maximum with the negative of the smallest value and returns the larger of the two. When abs_value is False the if-block will not be entered and the largest value max_value will be returned.

def largest(list_in, abs_value=False): max_value = max(list_in) if abs_value: min_value = min(list_in) return max(max_value, -min_value) return max_value

Test Calls

print(largest([1, 2, -3])) print(largest([1, 2, -3], False)) print(largest([1, 2, -3], True)) print(largest([1, 2, -3, 4], True))
